// デジタルと記録の永続性

紙の記録
劣化する。燃える。水に濡れる。でも、残った記録は数百年後も読める。福島大学に保存された葉書は、70年後も読める。
デジタルの記録
劣化しない。複製できる。でも、サービスが終了すれば消える。アカウントが削除されれば消える。サーバーが止まれば消える。
// 問い
小蘭が1949年に書いたバイナリは、帳面という「紙」に残った。それが70年後にデジタル化されてネットに浮上した。紙がなければ、デジタルもなかった。

// 付記 ── 小蘭の手記（高子沼にて）

私はITエンジニアだった。データは速く、正確に、遠くまで届けられるべきだと信じていた。

でも、この時代に生きてみて分かったことがある。

データは、誰かが意図して残さなければ消える。速さは永続を保証しない。クラウドに上げれば安全だと思っていた。でもクラウドの会社が潰れたら？サービスが終了したら？

帳面に書いたバイナリは、紙が残る限り消えない。障子に穿った穿孔パターンは、柱が残る限り消えない。支柱に刻んだ印は、鉄が残る限り消えない。

記録の形は関係ない。記録する意志が、記録を残す。

── 蛸川小蘭
