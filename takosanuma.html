T沼██施設跡　観覧車支柱の根元 ── 発見時の状況
タコのぬいぐるみと手帳が、支柱に寄りかかるように置かれていた

// 手帳より ── 最後の記録（復元）

── 平成██年　██月　夜

施設が閉まって、何年経ったのか、もう数えていない。

観覧車は、まだ立っている。回らないまま、ずっと立っている。私も、まだここにいる。

今夜、ここに来た理由は一つだ。記録を、隠す場所を決めた。

支柱の根元に、データを埋め込むように、帳面を置く。タコのぬいぐるみと一緒に。首のタグには「№0314」とある。黄ばんで、端が破れかけている。でも、番号は読める。

誰かが来たとき、分かるように。

── 同夜　続き

この印を、最後に刻む。

T湯の帳面に書いた。障子に穿った。そして今夜、支柱に刻む。三か所目だ。

0と1の羅列。●と○の並び。私はここにいた、という印。

帰れると思っていた　帰れなかった。でも、記録は残る。

猫塚清治は、存在した。私が記録した。

私は、存在した。████あなたが読んでいる。

// 穿孔パターン照合 ── 三か所の記録が一致

// 00番・帳面（1949年）
// 02番・障子（1950年代）
// 本ページ・支柱（閉園後）
── 三つのパターンは完全に一致する。
小蘭は50年以上にわたって、同じ印を残し続けた。
「私はここにいた」——その言葉が、時代を超えて繰り返されている。

// このページについて

この写真が撮影されたのは、記録が「発見された」時点だ。
誰が撮ったのか——小蘭自身が、最後にここを訪れた現代の自分に向けて残した可能性がある。

タコのぬいぐるみには、まだ名前がない。
小蘭は名前をつけなかった。それは、次に来る人間のためにとっておいた。
