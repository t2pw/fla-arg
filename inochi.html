// 冤罪はなぜ生まれたか ── M川事件の構造

// 自白の強要
最初の逮捕者は別件で拘束された少年だった。取り調べの中で「自白」させられ、その自白をもとに二十名が芋づる式に逮捕された。全員が公判で犯行を否認したが、裁判所は自白を重視して有罪とした。
// 証拠の隠蔽
被告たちの無実を示すアリバイ証拠が、捜査機関によって隠されていた。差し戻し審でこれが発覚し、逆転無罪の決め手となった。証拠を隠すことも、記録を消すことと同じだ。 → 猫塚清治が記録から消されたのも、同じ構造の中にある。
// メディアと世論の初期反応
事件直後、政府は「思想的底流」を問題にし、新聞・ラジオも当局の発表を追った。多くの国民が「労組員による犯行」と信じた。情報の非対称が、冤罪を成立させた。

// なぜ14年かかったのか

// 司法の壁
一審・二審は有罪を維持した。裁判所が自白と捜査当局の主張を信じ続けたからだ。証拠の再評価が行われるまでに、二度の上訴と最高裁の差し戻しが必要だった。
// 転機：広津和郎の裁判批判
作家・広津和郎が雑誌で裁判の矛盾を論証し始めたのは1953年。最高裁長官は広津の活動を「雑音」と呼んだ。しかしその声が、眠っていた世論を揺り起こした。文学者の言葉が、法廷を動かした。 → 07番の投書欄はこの「雑音」の一つを描いている。
// 守る会の力
全国に広がった支援運動「M川事件対策協議会」は、思想・階層を超えた市民が参加した。被告と家族・支援者の間では葉書・封書が飛び交い、福島大学には現在も葉書約一万点が保存されている。 → 08番のビラ、09番のスクラップブックはこの運動の記録をモチーフにしている。

// 手記③より ── 小蘭が「0314」に込めた意味

清治さんが死んだのは八月だ。桃の季節ではない。でも私がタグに書いた日付は三月十四日——桃の頃だ。なぜそうしたのか、今ならわかる。

帰れない日々の中で、帰れると信じられる季節が必要だったのだ。

そして清治さんには、夏の暗闇ではなく、花の中で眠っていてほしかった。

// このページが伝えたいこと

M川事件の冤罪は、一つの「悪意」だけで生まれたわけではない。自白を疑わなかった裁判官、当局の発表を追ったメディア、沈黙した市民——それぞれの「記録しなかった」選択が積み重なって、二十人の人生を奪いかけた。

そして今、SNSの時代に同じことが起きていないか。声の速さが変わっても、記録する意志がなければ真実は埋もれる。

猫塚清治という架空の人物は、「記録されなかった可能性」そのものだ。
