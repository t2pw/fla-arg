// 付記 ── 小蘭の手記（後期・高子沼にて）

ここに来て、五十年以上が経つ。

私はITエンジニアだった。情報の流れを設計することが仕事だった。データは速く、正確に、遠くまで届けられるべきだと信じていた。

でも、この時代に生きてみて、分かったことがある。

速さは、重さではない。

一通の葉書が被告の手に届いたとき、その重さはどれほどのものだったか。差出人の名前、折り目、滲んだインク。それが「ここに、あなたを信じている人間がいる」という証拠だった。

声の速さが変わっても、記録する意志がなければ、真実は埋もれる。

私がバイナリで書き残したデータが、七十年後にネットに浮上する。それはこの時代の葉書と、同じものだと思う。

届いてほしい人間に、届けばいい。それだけだ。

あなたは今、このアーカイブを読んでいる。

SNSで流れる情報を見るように、
数秒で次へ進むこともできる。

でも、ここまで読んだということは——

あなたは、記録した。

猫塚清治は、存在した。
あなたが読んだから。

// 現在の福島市とM川事件

松川事件から70年以上が経った現在も、福島市には事件の記憶が残っている。

松川運動記念館（福島市松川町）では、事件の経緯と市民運動の記録を展示している。福島大学松川事件資料室には約15,000点の一次資料が保管されている。

記録は残った。人々が残し続けたから。

あなたがこのアーカイブを見つけたように、誰かがいつか松川の記録を見つける。記録されたものは、消えない。
