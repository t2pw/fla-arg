24｜福島・桃の産地　H見山の現在

// H見山（花見山）── 現在も続く場所

// 史実
福島市のH見山（花見山）は、個人の農家が丹精込めて育てた花木の山を毎年春に一般開放している場所だ。桃・梅・桜・レンギョウが一斉に咲く。「桃源郷」とも呼ばれ、全国から観光客が訪れる。誰かに頼まれたわけでもなく、ただ「見てほしい」という気持ちで開放し続けている。 → 小蘭が「三月になれば桃が咲く。そうしたら帰れると思っていた」と書いたのはこの景色だ。
// 現在の福島市と桃
福島市は全国有数の桃の産地だ。福島の桃は「あかつき」「川中島白桃」などの品種で知られ、市の花にも制定されている。毎年夏、盆地の農園に桃が実る。
H見山 全景 ── 昭和三十二年春　桃・梅・桜　満開

// 小蘭が見た景色・あなたが見る景色

小蘭がT湯の宿の窓から眺めた桃の花は、今年も同じ場所に咲いた。

昭和二十六年の春に「来年の桃を、また待つことにした」と書いた小蘭は、それから何度その景色を見たのか。二十回か。三十回か。帰れないまま、桃の季節だけが繰り返された。

H見山の花は、誰かが記録しなくても、毎年咲く。でも、小蘭の記録がなければ、その景色がどれだけの重みを持っていたか——あなたには伝わらなかった。

記録が、景色に意味を与える。

現在の福島市では毎年三月下旬〜四月にH見山が一般開放される。入場無料。駐車場から徒歩数分。福島駅からシャトルバスが運行される期間もある。小蘭が「帰れると思っていた」季節は、今も誰でも訪れることができる。
