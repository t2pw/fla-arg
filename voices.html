// 葉書と封書が語るもの
拘置所の中の被告と、外の支援者が手紙を交わし続けた。一通の葉書が届くのに数日かかる時代に、それでも声は届き続けた。現在、福島大学松川事件資料室にその記録が保存されている。

// 声を上げた人々 ── 史実に基づく記録

広
広津 和郎（1891〜1968）
小説家・文芸評論家
「裁判所は論理を使わなかった。私はただ、論理を使ったに過ぎない。」
当初は「思想犯罪」と信じていたが、一審判決後に記録を精読して矛盾に気づく。雑誌『中央公論』などで裁判批判を展開し、当時の最高裁長官から「雑音」と呼ばれながらも書き続けた。 → 07番の投書欄「K鉄の同僚（自称）」が語る「声を上げ続けること」の原型はここにある。
家
被告の家族たち
妻・子・親・兄弟
死刑判決を受けた夫を持つ妻、父を獄中に持つ子。14年間、「犯人の家族」として生きながら、無実を訴え続けた。記録に残る葉書の多くは家族から被告へ、被告から家族へのものだ。 → 07番「保線作業員の妻」の投書はこうした家族の声をモチーフにしている。
市
名もなき市民たち
福島市民・全国の支援者
思想・職業・年齢を超えた市民が守る会に参加した。ガリ版ビラを刷り、新聞に投書し、集会に出た。誰に頼まれたわけでもなく、ただ「おかしい」と思ったから動いた。 → 08番のビラ、09番のスクラップブックに記録されているのはこうした市民の痕跡だ。
被
被告たち自身
国鉄・東芝の労働組合員20名
死刑判決を受けながら、最後まで無実を主張した。拘置所の中から詩を書き、意見を発表した。記録に残る被告たちの言葉は、絶望ではなく闘志に満ちている。
「私たちが黙れば、次は別の誰かが同じ目に遭う。」 ──（被告の言葉・趣意）

// アリバイ証拠の発覚
捜査機関が隠していたアリバイ証拠が、差し戻し審で明らかになった。これは弁護団と支援者が粘り強く記録を調査し続けた結果だった。記録を求め続けたから、隠された記録が出てきた。
// 広津批判の影響
広津和郎の裁判批判が世論を動かし、最高裁が差し戻しを決断する一因となった。「雑音」と呼ばれた声が、法廷を揺るがした。声の大きさではなく、声の継続が力になった。
