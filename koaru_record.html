事件当時：34歳 職業：K鉄（国鉄）保線作業員 状況：M川事件の夜、現場付近にいた。生存。 記録：被告リストなし。証人リストなし。公式記録上「存在しない」 その後：仕事を失い、記録の外で生き続けた。無罪にも有罪にもなれなかった。
架空の人物 ── M川（松川）事件をモチーフとした創作

// M川事件とは ── 史実に基づく記録

// 事件の概要
昭和二十四年（1949年）八月十七日午前三時九分、東北本線M川駅付近で上り旅客列車が脱線・転覆した。線路への破壊工作によるもので、乗務員三名が死亡、乗客ら八名が負傷した。 ※ このアーカイブでは「四人目の死者」として猫塚清治（架空）を設定している。
// 逮捕の経緯
事件から二十四日後、元国鉄線路工の少年が別件で逮捕され、取り調べの中で自供した。その自供をもとに芋づる式に逮捕が進み、国鉄・東芝両労働組合員など計二十名が起訴された。全員が犯行を否認したにもかかわらず、無実を示す証拠が捜査機関によって隠蔽されていた。
// 時代背景
1949年の夏は、占領下の日本で労働運動が高まっていた時期だった。M川事件の前後に「下山事件」「三鷹事件」が相次ぎ、当局は労働組合・共産党への弾圧に事件を利用しようとした。事件の真相は現在も不明のまま、時効が成立している。

// 裁判の経緯 ── 14年間・5度の審理

昭和24年
1949年
八月十七日未明、M川事件発生。乗務員三名死亡。 二十名が逮捕・起訴される。全員が無実を主張。
昭和25年
1950年
一審（福島地裁）：被告二十名全員有罪。うち死刑五名、無期懲役五名。 弁護団は即日控訴。
昭和28年
1953年
二審（仙台高裁）：有罪維持。死刑四名に。 被告・家族の訴えに、作家・広津和郎が雑誌で裁判批判を開始。
昭和34年
1959年
最高裁が審理を差し戻す。隠されていたアリバイ証拠が発覚。 全国組織「M川事件対策協議会」結成。支援者が全国に広がる。 → このアーカイブの守る会・新聞投書はこの時期の市民運動を描いている。
昭和36年
1961年
差し戻し審（仙台高裁）：被告全員無罪。 検察が上告。
昭和38年
1963年
最高裁：検察の上告を棄却。被告二十名全員の無罪確定。 事件発生から14年、5度の裁判を経た。 → 07番の投書欄「無罪が確定した」はこの年のこと。

// 世論を動かした人物 ── 広津和郎

// 文学者による裁判批判
小説家・広津和郎（1891〜1968）は、雑誌『中央公論』などで裁判の矛盾を徹底的に論証し続けた。当時の最高裁長官は広津の活動を「雑音」と呼んだ。しかしその「雑音」が世論を動かし、証拠の再調査につながった。 → 07番の投書欄に登場する「K鉄の同僚（自称）」のモデルの一つはこうした「声を上げた市民」である。

// このアーカイブにおける「猫塚清治」の位置づけ

猫塚清治は架空の人物だが、その「消され方」はM川事件の構造から導かれている。

実際の事件でも、捜査の都合によって証拠が隠された。二十名という数字が「正しい」と決められた。都合の悪い事実は記録されなかった。

猫塚清治は、そういった「記録されなかった可能性」を一人の人間として具体化した存在だ。

記録されなかったものは、存在しなかったことになる——本当にそれでいいのか。
